## 故障排除

### 问题1：文件选择后"已选择"列表不显示

**可能原因**：
1. JavaScript错误阻止代码执行
2. 事件监听器未正确绑定
3. DOM选择器错误

**解决步骤**：
1. 检查控制台是否有红色错误
2. 检查文件树中的文件项是否有 `data-path` 属性
3. 检查复选框是否有 `file-checkbox` 类名

### 问题2：计数不更新

**可能原因**：
1. `updateSelectedCount` 方法未调用
2. DOM元素选择器错误

**解决步骤**：
1. 在控制台输入 `app.selectedFiles.size` 查看内部状态
2. 检查 `elements.selectedCount` 是否正确指向元素

### 问题3：事件不触发

**可能原因**：
1. 事件监听器绑定失败
2. 事件冒泡被阻止

**解决步骤**：
1. 检查元素是否有事件监听器
2. 尝试直接点击复选框（不是文件行）


### 手动触发选择
```javascript
// 如果有文件复选框，手动触发选择
const firstFileItem = document.querySelector('.file-item');
if (firstFileItem) {
    const filePath = firstFileItem.dataset.path;
    const fileName = firstFileItem.querySelector('.file-name').textContent;
    app.toggleFileSelection(filePath, fileName, true);
    console.log('手动选择文件:', fileName);
}
```
