# Obsidian 讲义生成器 - 使用指南

## 概述

这是一个基于Python的Web应用程序，用于从Obsidian仓库生成自定义讲义。用户只需维护Obsidian仓库，就可以在Web界面上根据大纲选取内容，并控制是否显示【解析】或【注意】中的内容，最后生成属于自己的讲义。

## 主要特性

1. **目录浏览**：可视化浏览Obsidian仓库的目录结构
2. **文件选择**：选择要包含在讲义中的文件或文件夹
3. **内容控制**：控制是否显示【解析】和【注意】中的内容
4. **实时预览**：实时预览讲义内容
5. **多种格式**：支持HTML和Markdown格式输出
6. **搜索功能**：按文件名或内容搜索文件
7. **批量操作**：支持全选、清空等批量操作

## 快速开始

### 1. 环境准备

确保已安装Python 3.6或更高版本。

### 2. 安装依赖

```bash
cd lecture_generator
pip install -r requirements.txt
```

或者运行启动脚本自动安装：
- Windows: `start.bat`
- Linux/Mac: `./start.sh`

### 3. 配置检查

检查 `config.py` 文件，确保 `OBSIDIAN_REPO` 正确指向您的Obsidian仓库：
```python
# 默认配置
OBSIDIAN_REPO = os.path.join(os.path.dirname(os.path.abspath(__file__)), '..', 'obsidian_repo')
```

### 4. 启动应用程序

```bash
python app.py
```

或者使用启动脚本：
- Windows: 双击 `start.bat`
- Linux/Mac: `./start.sh`

### 5. 访问Web界面

打开浏览器，访问：http://localhost:5000

## 用户界面说明

### 左侧面板 - 文件选择

1. **文件树**：显示Obsidian仓库的目录结构
   - 文件夹：点击文件夹图标展开/折叠
   - 文件：勾选复选框选择文件
   - 图标说明：
     - 📁：文件夹
     - 📄：Markdown文件

2. **搜索功能**：
   - 输入关键词搜索
   - 可选择"搜索文件名"或"搜索内容"
   - 搜索结果中可直接选择文件

3. **已选择的文件**：
   - 显示当前选择的文件列表
   - 点击×图标可移除单个文件
   - 点击"清空选择"可移除所有文件

### 右侧面板 - 配置和预览

1. **生成选项**：
   - **内容显示**：
     - [ ] 显示【解析】内容
     - [ ] 显示【注意】内容
   - **输出格式**：
     - [ ] HTML格式
     - [ ] Markdown格式
   - **其他选项**：
     - [ ] 包含目录

2. **操作按钮**：
   - **预览讲义**：在下方预览区域显示讲义内容
   - **生成讲义**：生成文件并下载

3. **讲义预览**：
   - 实时显示讲义内容
   - 支持复制内容
   - 可清空预览

4. **信息面板**：
   - 显示当前配置信息
   - 显示选择的文件数量

## 使用步骤

### 第一步：选择文件

1. 浏览左侧的文件树，找到需要的文件
2. 勾选文件复选框，选择要包含的文件
3. 可以一次性选择多个文件
4. 可以在搜索结果中选择文件

### 第二步：配置选项

1. 根据需要设置内容显示选项：
   - 勾选"显示【解析】内容"：包含解析部分
   - 勾选"显示【注意】内容"：包含注意部分

2. 选择输出格式：
   - HTML格式：适合在浏览器中查看
   - Markdown格式：适合进一步编辑

3. 其他选项：
   - 勾选"包含目录"：在讲义开头添加目录

### 第三步：预览和生成

1. 点击"预览讲义"按钮查看内容
2. 检查预览内容是否符合预期
3. 点击"生成讲义"按钮生成文件
4. 在弹出的对话框中下载文件

## 配置说明

### 配置文件 (config.py)

```python
class Config:
    # Obsidian仓库路径
    OBSIDIAN_REPO = "路径/到/obsidian_repo"

    # 服务器配置
    HOST = '0.0.0.0'  # 监听地址
    PORT = 5000       # 监听端口
    DEBUG = True      # 调试模式

    # 输出目录
    OUTPUT_DIR = "路径/到/output"

    # 支持的文件扩展名
    MARKDOWN_EXTENSIONS = ['.md', '.markdown']
```

### 自定义配置

1. **修改Obsidian路径**：
   ```python
   OBSIDIAN_REPO = "/path/to/your/obsidian_vault"
   ```

2. **修改服务器设置**：
   ```python
   PORT = 8080  # 使用8080端口
   HOST = '127.0.0.1'  # 仅本地访问
   ```

3. **修改输出目录**：
   ```python
   OUTPUT_DIR = "/path/to/output"
   ```

## 故障排除

### 1. 应用程序无法启动

**问题**：`ModuleNotFoundError`
**解决**：运行 `pip install -r requirements.txt`

**问题**：`Address already in use`
**解决**：修改 `config.py` 中的 `PORT` 为其他端口

### 2. 无法找到Obsidian仓库

**问题**：`File not found`
**解决**：检查 `config.py` 中的 `OBSIDIAN_REPO` 路径

### 3. 文件内容显示不正确

**问题**：中文字符乱码
**解决**：确保Obsidian文件使用UTF-8编码

**问题**：特殊标签未过滤
**解决**：检查文件中的 `【解析】` 和 `【注意】` 标签格式是否正确

### 4. 无法下载文件

**问题**：浏览器阻止弹出窗口
**解决**：允许网站的弹出窗口

**问题**：防火墙阻止
**解决**：检查防火墙设置

## 扩展和定制

### 1. 添加新的内容标签

在 `obsidian_parser.py` 中修改 `_remove_section` 方法：
```python
def _remove_section(self, content, section_marker):
    # 添加对新标签的支持
    markers = ['【解析】', '【注意】', '【新标签】']
    for marker in markers:
        if marker == section_marker:
            pattern = re.compile(rf'{re.escape(marker)}.*?(?=\n【|$)', re.DOTALL)
            return pattern.sub('', content)
    return content
```

### 2. 修改输出格式

在 `lecture_generator.py` 中修改 `_generate_html` 或 `_generate_markdown` 方法来自定义输出格式。
